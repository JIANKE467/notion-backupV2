# 结构体（自定义数据类型）



struct：结构体（struct）是一种用户定义的数据类型，它允许你将不同类型的数据组合在一起。



## 先定义

定义结构体结构（定义结构体名）

![image](./../assets/img_1e75a2dd-8276-8101-a0ff-ef247645aefe.png)

20250407211430127-1

```c
struct 结构体名
{
  int 成员名1；//成员1
  float 成员名2；//成员2
  数据类型 成员名3；//成员3
.....
};
```

## 在定义

定义结构体变量（定义结构体变量名）

![image](./../assets/img_1e75a2dd-8276-8173-878d-ee21b4fd9e7a.png)

20250407212847555

```c
struct 结构体名 结构体变量名;
```

## 输入值

为结构体变量输入值

```c
结构体变量名={值1,值2,值3};
```

也可以直接初始化结构体变量

```c
struct 结构体名 结构体变量名={值1,值2,.值3}
```

## 访问

访问结构体成员

使用点操作符 . 来访问结构体的成员

如：

```c
printf("%d%d%d",结构体变量名.成员名);
```
