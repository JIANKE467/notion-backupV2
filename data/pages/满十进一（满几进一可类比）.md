# 满十进一（满几进一可类比）

利用for循环嵌套：(三层for分别控制 个 十 百 位）

例子：打印出100到999的数（体现满10进一的思想）

```c
#include<stdio.h>
int main()
{
 int bai,shi,ge,x;
 for(bai = 1; bai <= 9; ++bai)
   { // 百位数字从1到9
   for(shi = 0; shi <= 9; ++shi)
     { // 十位数字从0到9
     for(ge = 0; ge <= 9; ++ge)
       { // 个位数字从0到9
        x=bai*100+shi*10+ge;
        printf("%d\n",x);
       }
     }
   }
return 0;
}
```
